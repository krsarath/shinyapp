# Contributing to Shiny Vista

## Reporting bugs
Please open a GitHub issue describing the bug and steps to reproduce it.

## Suggesting features
Open a GitHub issue with the label "enhancement".

## Development setup
1. Clone the repository
2. Install R (v4.4+) and required packages (see README)
3. Run shiny::runApp() to launch locally
